# 🍳 냉장고를 털어라! (Fridge Recipe App with Gemini AI)

대학생 자취생들을 위해 냉장고 속 남은 식재료 조합을 분석하고, 구글 **Gemini AI**를 활용하여 실시간으로 맞춤형 가성비 초간단 레시피를 제안해주는 1페이지 웹 애플리케이션입니다.

본 프로젝트는 **GitHub Pages**에 최적화되어 빌드 과정 없이 정적 웹페이지로 즉시 배포 및 실행이 가능합니다.

---

## ✨ 주요 기능
1. **식재료 선택 멀티 칩 UI**: 채소, 육류, 소스 등 카테고리별 체크박스로 냉장고 재료를 손쉽게 추가/삭제합니다.
2. **실시간 식재료 검색 및 직접 추가**: 목록에 없는 특이한 재료도 수동으로 바로 추가해 매칭할 수 있습니다.
3. **로컬 브라우저 저장소(localStorage) 연동**: 새로고침을 하거나 나중에 재접속해도 이전에 선택한 재료와 API 키가 안전하게 보존됩니다.
4. **Gemini 1.5 Flash API 연동**: 구글의 초고속 경량 AI 모델을 브라우저 환경에서 직접 호출하여 요리 팁을 포함한 레시피 3종을 실시간으로 창작합니다.
5. **안전한 API 키 관리**: 하드코딩 없이 사용자가 화면상에서 직접 API 키를 입력하는 방식으로, GitHub Public 저장소에 배포해도 API 키 유출 염려 없이 안전하게 공유할 수 있습니다.

---

## 🚀 시작하기 & 사용법

### 1. 웹사이트 실행
*   배포된 GitHub Pages 링크 또는 로컬 환경의 `index.html` 파일을 더블클릭하여 브라우저에서 실행합니다.

### 2. Gemini API 키 발급 및 입력
1. [Google AI Studio](https://aistudio.google.com/)에 구글 계정으로 로그인합니다.
2. **Create API Key**를 클릭하여 고유 API 키를 무료로 발급받습니다.
3. 웹페이지 상단 우측의 **🔑 Gemini API Key** 입력창에 복사한 키를 넣고 **[저장]** 버튼을 클릭합니다.
4. (저장된 키는 사용자의 로컬 브라우저에만 저장되며 외부 서버나 타인에게 전송되지 않습니다.)

### 3. 냉장고 파먹기 시작!
1. 냉장고에 있는 재료들을 체크합니다.
2. 목록에 없는 재료는 직접 검색창 오른쪽에서 텍스트로 추가합니다.
3. **[🍳 AI 레시피 매칭!]** 버튼을 클릭하면 AI 요리사가 만들어내는 꿀팁 가득한 요리 단계를 보실 수 있습니다.

---

## 🛠️ 기술 스택
*   **Frontend**: HTML5, JavaScript (ES6+)
*   **Styling**: Tailwind CSS (via Play CDN)
*   **AI Integration**: Google Gemini 1.5 Flash API (REST API)
*   **Design**: Modern Organic Cream Theme (모던 오가닉 크림 테마)
